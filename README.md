This is the Linux build for Immortal Combat, a Fighting Game based on my interactions on Christian Twitter

In order to install this, download and run the installer.py script.

Thank you very much to @Reese_Defton on X for helping build the installer script.


Game Controls


Spacebar: Confirm, Jump

Z: Ground attack (Slightly more powerful, cannot be used out of a jump)

X: Back, Special attack (Slightly weaker than a ground attack, but can be used out of a jump)

C: Super attack (When super meter is full)

Directional Buttons: Move around


How to play:


Each player has three lives. If your health is depleted you will lose a life. Your super meter fills up every time you successfully land an attack. When it is completely full you can use your super attack. The person who makes the other lose all of their lives first wins.
